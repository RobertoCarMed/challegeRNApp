## Usage

### Installing software requirements

#### Required

- Node
- Emulator or Android device
- IOS Emulator

#### Optional

- JDK 11
- Android Studio

### Installing project dependencies

```
yarn
```

### Running the project

#### Any supported platform

```
yarn start
```

#### Android

```
yarn android
```

#### iOS

```
yarn ios
```

## Technologies used

- Node
- TypeScript
- React Native
